# Tic Tac Toe

[![Netlify Status](https://api.netlify.com/api/v1/badges/f9038247-5d90-470f-bc63-163d637a2f91/deploy-status)](https://app.netlify.com/sites/topher-tictactoe/deploys)

### Technologies used:

- Typescript
- ReactJS
- Vanilla CSS

### Functionality:

- Win, Lose, Draw Condition
- Undo Move Button
- Restart Game Button
- Game History
- Screen shake on bad input (CSS effects)
- Game Over Modal
