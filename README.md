# Tic Tac Toe

### Technologies used:

- Typescript
- ReactJS
- Vanilla CSS
- No external Libaries

### Functionality:

- Win, Lose, Draw Condition
- Undo Move Button
- Restart Game Button
- Game History
- Screen shake on bad input (CSS effects)
- Game Over Modal
